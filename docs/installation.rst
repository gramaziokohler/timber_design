********************************************************************************
Installation
********************************************************************************

Stable
======

Stable releases are available on PyPI and can be installed with pip.

.. code-block:: bash

    pip install timber_design


Latest
======

The latest version can be installed from local source.

.. code-block:: bash

    git clone https://github.com/gramaziokohler/timber_design.git
    cd compas_tna
    pip install -e .


Development
===========

To install `timber_design` for development, install from local source with the "dev" requirements.

.. code-block:: bash

    git clone https://github.com/gramaziokohler/timber_design.git
    cd compas_tna
    pip install -e ".[dev]"
